## 1. 基本语法
- 区分大小写：与 Java 一样，变量名、函数名以及其他一切东西都是区分大小写的

- 每行结尾的分号可有可无

- 注释：

  - 单行注释：// 注释内容

  - 多行注释：/* 注释内容 */

## 2. 3种输出语句

| api              | 描述             |
| ---------------- | ---------------- |
| window.alert()   | 警告框           |
| document.write() | 在HTML 输出内容  |
| console.log()    | 写入浏览器控制台 |
